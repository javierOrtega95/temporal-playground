# Temporal Playground

Interactive playground to explore the JavaScript Temporal API:
- Time zones
- Durations
- Parsing and formatting
- Date arithmetic

## Tech Stack
- React + TypeScript
- TailwindCSS
- Vite + SWC
- Temporal Polyfill (for older browsers / Node versions)

## Getting Started (Node >=20 + pnpm)

```bash
fnm use 20
corepack enable
corepack prepare pnpm@latest --activate
pnpm install
pnpm dev
```
